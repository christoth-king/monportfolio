# 🚀 Dev_sco — Portfolio Développeur Web & UI Designer

Bienvenue sur mon portfolio personnel ✨  
Ce projet présente mes compétences, mes réalisations et mes services en tant que développeur web & designer UI.

---

## 📌 Aperçu du projet

Ce site est un **portfolio moderne et responsive** conçu pour mettre en valeur :

- 👨‍💻 Mes compétences techniques
- 🎨 Mon sens du design UI/UX
- 📂 Mes projets réalisés
- 📞 Un système de contact direct (email & WhatsApp)

---

## 🛠️ Technologies utilisées

Le projet a été développé **sans framework**, uniquement avec les fondamentaux du web :

- HTML5 — Structure du site  
- CSS3 — Design, animations et responsive  
- JavaScript (Vanilla) — Interactions dynamiques  
- PHP — Gestion du formulaire (WhatsApp)  
- MySQL — (dans certains projets liés)

---

## ✨ Fonctionnalités principales

- 🎯 Design moderne avec effets visuels (glow, animations)
- 📱 Responsive (mobile, tablette, desktop)
- 🧭 Navigation fluide avec scroll smooth
- 🎬 Animations au scroll (Intersection Observer)
- 💡 Effet curseur lumineux interactif
- 📊 Barres de progression des compétences
- 📂 Section projets dynamique
- 📩 Formulaire de contact fonctionnel

## 🚀 Installation & utilisation

1. Cloner le projet :
git clone https://github.com/ton-username/ton-repo.git

2. Ouvrir le fichier index.html dans un navigateur

3. (Optionnel) Pour le formulaire PHP :
- Installer XAMPP
- Placer le projet dans htdocs
- Lancer Apache
- Accéder via http://localhost/ton-projet
