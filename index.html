À propos

Passionné par le web
depuis toujours

Je suis Christ Oth, étudiant en BTS IDA et développeur web freelance. Depuis mes débuts, j'ai une passion pour la création d'interfaces web qui allient performance technique et esthétique soignée.

Ce portfolio a été entièrement conçu et développé par mes soins, en utilisant HTML, CSS et JavaScript pur — sans framework — pour démontrer ma maîtrise des fondamentaux du web.

Mon approche : chaque projet est une opportunité de livrer une expérience utilisateur mémorable, responsive et optimisée.
